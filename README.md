# Bertram Labs Coding Challenge
---
## Getting Started

#### Built with:
* `Python (Version 3.10.2)`
* `SQLite`

#### Installation:
1. Clone the repo:
```sh
git clone https://github.com/DhrutikS257/Bertram-Coding-Challenge
```
2. Run the program:
```sh
    cd Bertram-Coding-Challenge
    make main
```
